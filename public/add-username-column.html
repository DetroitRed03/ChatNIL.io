Step-by-step migration to add usernames to athlete profiles

Step 1: Add Column via Supabase Dashboard

Since we can't execute DDL through the API, please:

  1. Open Supabase Table Editor
  2. Select the users table
  3. Click "Add Column" and create:
    • Name: username
    • Type: text
    • Check "Is Unique"
    • Nullable: Yes
  4. Click "Save"

OR use the SQL Editor:

  1. Go to SQL Editor
  2. Paste this SQL: 
    ALTER TABLE users ADD COLUMN username TEXT UNIQUE;
CREATE INDEX idx_users_username ON users(username);
  3. Click "Run"
