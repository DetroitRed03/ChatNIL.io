Troubleshooting
If the dropdown is empty:
  • Check that Step 2 (seeding) completed successfully
  • Open browser console (F12) and check for errors
  • Verify API endpoints return data:
    curl http://localhost:3000/api/demo/fmv/athletes
If you see 500 errors:
  • Make sure Step 1 (migration) completed without errors
  • Check Supabase dashboard to verify tables exist
  • Ensure your .env.local has valid Supabase credentials
If FMV scores don't display:
  • FMV calculation must have been run previously (migration 015)
  • Check that athlete_fmv_data table has data
  • Run: npx tsx scripts/calculate-fmv-all.ts if needed
